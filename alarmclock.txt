#알람 시계

시 > a
분 > b

현재 시각에 45분을 빼는 연산을 하시오.
a은 (0 <= a < 24) 의 값만 가질 수 있다.
b은 (0 <= b < 60) 의 값만 가질 수 있다.
예) 10시 30분 - 45분 = 9시 45분
ex) int b = b - 45;

1. 만약 b에 45분을 뺀 값이 0보다 작을 때 a는 1 감소한 후, 임의의 변수 c를 만들어 b-45한 값을 저장한 후, c + 60을 해준다. 이후 c값을 b로 치환하면 a는 1이 감소되어있고, b는 45분이 감소된 값을 나타낸다(c).
if(b - 45 < 0)
{
int c = b - 45;
a--;
c = c + 60;
b = c
}

2. 만약 분이 0보다 크다면 시간은 감소하지 않고, 현재 분 값만 45만큼 감소한다
위의 if문에 이어서
else
{
b = b - 45;
}
